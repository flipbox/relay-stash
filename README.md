This package provides a [Stash](https://github.com/tedious/Stash) middleware for [Relay](http://relayphp.com/).

## Installation

To install, use composer:

```
composer require flipboxdigital/relay-stash
```

## Testing

``` bash
$ ./vendor/bin/phpunit
```
